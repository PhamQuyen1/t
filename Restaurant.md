# Project: Restaurant
Design and implement RESTful web service for a restaurant in order to support menu and bill order management.
# 📁 Collection: Menu 


## End-point: Get Menu with pagination and search
Get page menu
### Method: GET
>```
>localhost:8080/menu?page=1&keyword=Xiaolongbao
>```
### Query Params

|Param|value|
|---|---|
|page|1|
|keyword|Xiaolongbao|
